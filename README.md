# Flint Town Urban Ballroomers

## Description

This website is the official website of the Flint Town Urban Ballroomers, an organization that hosts and teaches dancers in this space.

### Technologies Used

* HTML
* CSS
* JavaScript
* React
* Responsive Design  

---

<img src="./src/assets/htmlandcss.png" alt="Picture of HTML and CSS logos" width="100" height="50"> **HTML, CSS, Flexbox**
  
This project was made utilizing a combination of coding with all of these technologies.

<img src="./src/assets/java.png" alt="Picture of JavaScript logo" width="50" height="50"> **JavaScript**

This project was made utilizing JavaScript. Several functions and variables were used to execute actionable features.

<img src="./src/assets/react.png" alt="Picture of JavaScript logo" width="70" height="50">**React**

This project utilized React accompanied with Hooks to manage state.

**Responsive Design**

This project required functionality of the website at all resolutions to include more than 1400 pixels all the way down and adaptive to a screen size of 350 pixels.
